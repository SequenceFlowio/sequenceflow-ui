# SequenceFlow go-to-market

This directory is the canonical source for positioning, personas, paid-media execution, and ad copy for the SequenceFlow email-support product.

## Source of truth

- `brand/BRAND_DNA.md`: positioning, promise, proof, voice, visual system, and claim guardrails.
- `personas/`: the three buying audiences and their jobs, objections, triggers, and messages.
- `strategy/PAID_MEDIA_100_EUR_PER_DAY.md`: budget, measurement, launch sequence, and decision rules.
- `campaigns/`: campaign and ad-set setup, complete copy, URLs, UTMs, exclusions, and creative briefs.
- `creatives/`: editable SVG ad masters in the SequenceFlow visual system.

The public routes used by campaigns are:

| Audience | Landing page |
| --- | --- |
| General / high intent | `/` |
| Webshops | `/for/webshops` |
| Customer-service teams | `/for/customer-service-teams` |
| E-commerce founders | `/for/ecommerce-founders` |
| Pricing intent | `/pricing` |

Do not use the older “AI Operating Systems” positioning for this product. It is too broad for the current application and weakens message-to-market fit.
